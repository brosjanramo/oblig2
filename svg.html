SVG

SVG står for Scalable Vector Graphics. Vi kan lage 2D vectorbaserte bildeformater. Vi kan bruke JavaScript og CSS til å manipulere SVG. Vi kan bruke SVG til å lage ikoner, animasjoner, logoer, grafer, illustrasjoner. SVG mister ikke sin kvalitet når vi forstørrer størrelsen. Vi kan bruke eksterne programmer til å lage SVG, som Inkscape og adobe illustrator eller kode det direkte inn i html. Slik som det er gjort under.
