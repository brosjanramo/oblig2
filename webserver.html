Webservere

Hva er en webserver?

Webserver er en maskin med programvare som håndterer nettsider. Programvarer som Apache, Linux og wampserver blir ofte brukt. Hovedoppgaven til en webserver er å bringe oss data vi spør etter på internett. Vi bruker nettlesere til å få fremvist data fra webservere og protokollene som brukes for å hente data på nettleserne er HTTP(Hyper Text Transfer Protocol) eller HTTPS(Hyper Text Transfer Protocol Secure). En Webserver trenger ikke å være på internett. Det kan også bli brukt lokalt på printere, rutere og Webkamera.

Wamp Server

Wamp Server er blir brukt av operativsystemet Windows.

MAMP

MAMP blir brukt av operativsystemet macOS.

LAMP

LAMP blir brukt av operativsystemet linux.
